## 소개

## 본인 구현    

* **회원    [ 가입/탈퇴, 개인정보 조회/수정 ]**
* **관리자  [ 회원목록, 정보조회, 회원삭제 ]**     

    
    
    
## 기능 정리       

* 회원 가입/탈퇴
* 개인정보 조회/수정
* 로그인 후 게시판 글쓰기
* 제품 장바구니 담기
* 로그인 후 제품 찜하기 / 제품 구입
* 관리자모드 회원정보 조회
